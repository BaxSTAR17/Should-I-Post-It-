Should I Post It? (SIP for short) is an open-source webapp made to predict a draft post's reception on Social Media through the use of Predictive Analytics

## Key Features
- Users are able to input their supposed draft post into the webapp's interface
- Analysis of the draft post using a pre-trained model for predictive analytics
- The webapp's model will then be able to make a probability distribution of how well the post will be received by the public.
- Showcases data visualization and insights to users if it would be worth it to post or not
